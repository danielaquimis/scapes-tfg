# Environmental Sound Generation and Evaluation: Exploring Architectural Variations of the SCAPES Model

This repository contains the code, evaluation results, audio examples, and supplementary material associated with the Bachelor's Thesis:

**Environmental Sound Generation and Evaluation: Exploring Architectural Variations of the SCAPES Model**

Universitat Pompeu Fabra (UPF), 2026

## Overview

This work investigates several architectural modifications to SCAPES (Semantically Conditioned Autoregressive Prior for Environmental Sounds), a lightweight generative model for environmental audio synthesis.

The evaluated systems are:

* Baseline SCAPES
* Proposal 1: Structure Feature Conditioning
* Proposal 2: GAN-Based Adversarial Regularization
* Proposal 3: Phase Regularisers for Atom-Level Coherence

## Repository Structure

* `audio/` — Original and generated audio samples.
* `code/` — Implementation of the proposed modifications and evaluation scripts.
* `results/` — FAD, KAD, and training-loss results.
* `figures/` — Figures used in the thesis.
* `DATASET_SOURCES.md` — Attribution and licensing information for Freesound recordings.

## Dataset

The evaluation dataset was constructed from recordings obtained from Freesound.

Attribution information, source identifiers, and license details are provided in `DATASET_SOURCES.md`.
